Objetivo del programa: 
Poder generar una agenda que permmita guardar la información (nombre completo, número de teléfono y correo electrónico) de distintos contactos, las funciones de este programa van a permitir mostrar en pantalla la información actualmente guardada, ordenar la agenda según el usuario desee y modificar los datos almacenados de algún contacto.

Actuales limitaciones: 
Sólo se puede ordenar por A-Z o Z-A.
No se puede modificar la cantidad de contactos ni el tope de los strings una vez establecidos al comienzo de la ejecución.
Una vez se detiene el programa, los datos se pierden.
